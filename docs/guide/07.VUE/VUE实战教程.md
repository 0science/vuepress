## 项目四：响应式数据绑定

目标：成功使用响应式绑定一条数据

### ref()
数据转换成响应式数据
```
响应式数据 = ref(数据)
响应式数据.value = 数据
```

### reactive()
创建响应式对象或数组
```
响应式数据 = reactive(对象或数组)
响应式数据.属性 = 数据
```

### toRef()
将响应式对象中的属性转换成响应式数据
```
响应式数据 = toRef(响应式对象, '属性名')
响应式数据.value = 数据
```

### toRefs()
将响应式对象中的所有属性转换成响应式数据
```
响应式数据 = toRefs(响应式对象)
响应式数据.属性 = 数据
```

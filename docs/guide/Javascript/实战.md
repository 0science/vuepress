## JavaScript实战案例

### 输出hello world

既在页面中输出hello world，也在控制台输出hello world

```
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>hello world</title>
</head>
<body>
    <script>
        document.write('hello world') // 输出到页面
        console.log('hello world') // 输出到控制台
    </script>
    </script>
</body>
</html>
```

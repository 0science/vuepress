
### 语义化标签的好处

1. 提高用户体验
2. 有利于SEO，方便搜索
3. 具有可读性
4. 代码结构更好

### 重排和重绘

重排：当页面元素位置发生变化，浏览器会重新计算页面元素位置，然后重新渲染页面。

1. 增删dom元素
2. 位置改变
3. 尺寸改变
4. 内容改变
5. 第一次渲染


重绘：当页面元素外观发生变化，浏览器会重新渲染页面元素。

1. 布局未变，外观改变
2. 样式改变

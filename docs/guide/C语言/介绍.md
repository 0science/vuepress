

## 第一个C程序

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```
输出`Hello, World!`

## 1.程序语言
| 程序语言 | 特点                                                 | 举例                      |
| -------- | ---------------------------------------------------- | ------------------------- |
| 机器语言 | 低级语言，0和1组成的二进制代码，计算机直接识别。     | 如：`10110000 01100001`   |
| 汇编语言 | 面向机器的程序设计语言，用英文和字符代替二进制代码。 | 如：`MOV AL, 61h`         |
| 高级语言 | 语法类似英文，贴近自然语言。                         | 如：C语言、Python、Java等 |
## 2.特点
| 优点       | 缺点           |
| ---------- | -------------- |
| 代码量小   | 代码实现周期长 |
| 运行速度快 | 平台库依赖较多 |
| 功能丰富   |                |

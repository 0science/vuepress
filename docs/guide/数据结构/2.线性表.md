#### 顺序存储与链式存储的优缺点
|      | 顺序表               | 链表                           |
| ---- | ------------------ | ------------------------------ |
| 特点 | 逻辑与物理地址统一 | 元素存放随意，有指针和和结点值 |
| 优点 | 存储密度高         | 增删不需要移动元素             |
| 缺点 | 增删需要移动元素   | 存储密度低                     |
| 增删 | O(n)                   |     O(1)                           |
| 查     |   O(1)                 |     O(n)                           |



#### 如何选用存储结构
选顺序表：长度变化不大，主要是查找
选链表：长度变化大，主要是增删
